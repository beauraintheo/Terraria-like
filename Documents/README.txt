Controles pour notre jeu : 


Déplacements : 

[Z] / [Flèche haut]  =  Sauter
[Q] / [Flèche gauche]  =  Déplacement gauche
[D] / [Flèche droite]  =  Déplacement droite

Inventaire : 

De [1] à [8] OU de [F1] à [F8] si les premières ne fonctionnent pas  =  Accéder à l'item dans l'inventaire à la position souhaitée 

[1]  =  Epée
[2]  =  Pioche
[3]  =  Terre
[4]  =  Pierre
[5]  =  Bois
[6]  =  Charbon
[7]  =  Fer
[8]  =  Fleur

Si l'épée est sélectionée : [Clic gauche]  =  Taper 

Si la pioche est sélectionée : [Clic gauche]  =  Miner

Si un bloc est sélectioné :  [Clic droit]  =  Poser


Warning : Les végétaux tels que les fleurs ou les herbes peuvent etre cassées à la main !



PS : Votre joueur ne peut casser / poser un bloc qu'à un bloc de distance. Pour augmenter cette distance, veuillez aller dans la classe Joueur à la méthode "peutPoserCasserBloc(Coordonnees coord)" et modifier les "1" en la valeur souhaitée.
